## Doel

Elk teamlid heeft een leerpad dat past bij zijn rol en groeistadium. Training is gekoppeld aan believability levels: je groeit van Beginner (★) naar Master (★★★★★) door gerichte ontwikkeling.

---

## Leerpad Structuur

### Per Believability Level

```
★☆☆☆☆ BEGINNER (Week 1-4)
│  Doel: Begrijp de basics, volg processen
│  Methode: Lezen, observeren, mentor
│  Toets: Onboarding kennistoets
│
★★☆☆☆ BASIS (Maand 2-3)
│  Doel: Voer zelfstandig taken uit
│  Methode: Doen onder begeleiding, feedback loops
│  Toets: Domein kennistoets (basis)
│
★★★☆☆ GEVORDERD (Maand 4-6)
│  Doel: Los complexe problemen zelfstandig op
│  Methode: Eigenaarschap, cross-team projecten
│  Toets: Scenario-based assessment
│
★★★★☆ EXPERT (Maand 6+)
│  Doel: Wordt raadpleegbaar, begeleid anderen
│  Methode: Mentoring, architecture decisions, kennisdeling
│  Toets: Peer review + track record analyse
│
★★★★★ MASTER (Maand 12+)
│  Doel: Definieer best practices, leidt het domein
│  Methode: Innovatie, strategische bijdragen
│  Toets: Impact analyse + domein leiderschap
```
