## 2. Owned Channels

### 2.1 Website/Blog

**Doel:** SEO traffic, autoriteit opbouwen

**Acties:**
- 2-3 artikelen per week publiceren
- Pillar pages maandelijks updaten
- Interne links optimaliseren
- CTA's A/B testen

**KPI's:**
- Organische sessies
- Tijd op pagina
- Blog → Connect conversie

### 2.2 LinkedIn (Bedrijf)

**Doel:** Brand awareness, B2B bereik

**Frequentie:** 3x per week

**Content mix:**
- 40% Educatie/tips
- 30% Thought leadership
- 20% Product nieuws
- 10% Behind the scenes

**Best practices:**
- Post tussen 8-10 uur
- Geen externe links in eerste comment
- Vraag om interactie
- Reply op comments binnen 2 uur
- Gebruik max 5 hashtags

**Hashtags:**
```
#AIboekhouding #ExactOnline #MKB #Ondernemen #Boekhouding
#Automatisering #FinTech #ZZP #Accountancy #AI
```

### 2.3 LinkedIn (Founders)

**Doel:** Personal brand, thought leadership

**Frequentie:** 3-5x per week

**Content types:**
- Maandag: Weekstart / motivatie
- Dinsdag: Tip of lesson learned
- Woensdag: Building in public
- Donderdag: Industrie reflectie
- Vrijdag: Week recap / vraag

**Format templates:**

**Hook + Story:**
```
[Provocerende eerste zin]

Vorige week [gebeurtenis].

Dit leerde ik:
→ [Les 1]
→ [Les 2]
→ [Les 3]

[Vraag aan volgers]
```

**Carousel:**
```
Slide 1: Hook vraag
Slide 2-5: Content/tips
Slide 6: CTA + volg mij
```

### 2.4 Email Nieuwsbrief

**Doel:** Nurturing, retention

**Frequentie:** Maandelijks (later bi-weekly)

**Structuur:**
```
Subject: [Emoji] [Benefit-driven headline]

Preview text: [Teaser]

---

Hey [Naam],

[Intro - 2 zinnen over thema]

## 1. [Tip van de maand]
[Korte uitleg + link naar artikel]

## 2. [Product update]
[Wat is nieuw]

## 3. [Gelezen/gezien]
[Interessante link met context]

---

Tot volgende maand!
[Naam]

P.S. [Soft CTA of vraag]
```

**Growth tactics:**
- Lead magnet op blog (checklist, template)
- Exit intent popup
- Content upgrade per artikel
- Webinar registraties

### 2.5 GitHub

**Doel:** Developer community, backlinks

**Acties:**
- README optimaliseren voor SEO
- Contributing guidelines
- Changelog bijhouden
- Issues actief beantwoorden

## 6. Content Repurposing

### 6.1 Blog → Multi-Format

| Origineel | Repurpose naar | Platform |
|-----------|----------------|----------|
| Pillar page | E-book/PDF | Lead magnet |
| Pillar page | Webinar | YouTube/Live |
| How-to post | Carousel | LinkedIn |
| How-to post | Thread | Twitter/X |
| How-to post | Short video | TikTok/Reels |
| Listicle | Infographic | Pinterest/LinkedIn |
| Case study | Quote graphics | Instagram |
| Tutorial | Screencast | YouTube |
| FAQ | Story series | Instagram |

### 6.2 Repurposing Workflow

**Stap 1: Identificeer evergreen content**
- Pillar pages
- Best performing blogs
- Tutorials

**Stap 2: Creëer assets**
- Extract key points (3-5 per artikel)
- Maak visual templates
- Script videos

**Stap 3: Schedule**
- Spread over meerdere weken
- Niet alles tegelijk posten
- Varieer platforms

### 6.3 Content Atomization Voorbeeld

**1 Pillar Page "AI Boekhouding Gids" wordt:**

```
Week 1:
- LinkedIn post: "3 mythes over AI boekhouding"
- Twitter thread: 10 dingen die AI wel/niet kan

Week 2:
- Carousel: "AI boekhouding in 5 stappen"
- Email: Samenvatting + link

Week 3:
- Quote graphic: Kernboodschap
- YouTube short: 60 sec explainer

Week 4:
- Podcast mention/pitch
- Guest post met nieuwe hoek
```

## 7. Launch Playbook

### 7.1 Nieuw Artikel Checklist

**Dag 0 (Publicatie):**
- [ ] Artikel live
- [ ] Social media posts klaar
- [ ] Email aan subscribers (indien major)

**Dag 0 (binnen 2 uur):**
- [ ] LinkedIn post (personal)
- [ ] LinkedIn post (company)
- [ ] Twitter post
- [ ] Relevante LinkedIn groepen (1-2)

**Dag 1:**
- [ ] Reddit waar passend
- [ ] Email aan relevante contacten
- [ ] Interne links toevoegen vanuit oude content

**Week 1:**
- [ ] Outreach naar influencers
- [ ] Pitch naar nieuwsbrieven
- [ ] Repurpose planning starten

**Maand 1:**
- [ ] Performance check
- [ ] Optimize title/meta indien nodig
- [ ] Plan update/refresh

### 7.2 Major Launch Playbook

**Voor grote pillar pages, features, of nieuws:**

**T-7 dagen:**
- [ ] Teaser content op social
- [ ] Exclusive preview aan key contacts
- [ ] Perslijst voorbereiden

**T-1 dag:**
- [ ] Final review
- [ ] Alle assets klaar
- [ ] Team briefing

**Launch dag:**
- [ ] Publiceren vroeg (8:00)
- [ ] Persberichten versturen
- [ ] All-hands social push
- [ ] Email blast
- [ ] Monitoren en reageren

**T+1 week:**
- [ ] Performance review
- [ ] Volgende wave social
- [ ] Outreach ronde 2
