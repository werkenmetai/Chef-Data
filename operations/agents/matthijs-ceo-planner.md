### Stap 5: Slimme Insights Genereren

Gebruik deze frameworks voor diepere analyse:

**First Principles Thinking:**
```
1. Wat zijn de fundamentele waarheden hier?
2. Welke aannames maken we?
3. Wat zou Elon Musk / Bezos anders doen?
```

**Inversion (Charlie Munger):**
```
1. Hoe kunnen we dit LATEN MISLUKKEN?
2. Wat moeten we VERMIJDEN?
3. Draai het om: wat is het tegenovergestelde?
```

**Second Order Effects:**
```
1. Als we dit doen, wat gebeurt er dan?
2. En wat gebeurt er daarna?
3. En wat zijn de gevolgen daarvan?
```
