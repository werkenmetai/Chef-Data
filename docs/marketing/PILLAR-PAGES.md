## SEO Optimalisatie per Pillar

### On-Page Checklist

Voor elke pillar page:

- [ ] Title tag: keyword + benefit (max 60 chars)
- [ ] Meta description: hook + CTA (max 155 chars)
- [ ] H1: bevat primair keyword
- [ ] H2's: bevatten secundaire keywords
- [ ] URL: kort, keyword-rijk
- [ ] First 100 words: primair keyword
- [ ] Alt tekst: alle afbeeldingen beschrijvend
- [ ] Interne links: min. 5 naar cluster artikelen
- [ ] Externe links: 2-3 naar autoritaire bronnen
- [ ] FAQ sectie: vraag-gebaseerde keywords
- [ ] Inhoudsopgave: met anchor links
- [ ] Leestijd: prominant tonen
- [ ] Laatst bijgewerkt: datum tonen
- [ ] CTA: minimaal 2 per pagina
- [ ] Schema: Article + FAQ markup
