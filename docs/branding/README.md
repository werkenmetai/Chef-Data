## Quick Reference

### Kernidentiteit
**Betrouwbaar – Toegankelijk – Vakkundig**

### Gouden Regels

1. **Altijd "je/jij"** - nooit "u"
2. **Actief schrijven** - "Je krijgt" niet "Wordt verkregen"
3. **Onderbouw claims** - cijfers, voorbeelden, bewijs
4. **Eerlijk over beperkingen** - "Ik kan alleen lezen"
5. **Laagdrempelige CTA's** - "Probeer gratis" niet "Koop nu"

### De Stem
> Een ervaren financial controller van begin 40: professioneel maar niet stijf, deskundig maar niet neerbuigend, behulpzaam maar niet opdringerig.

## Per Kanaal

| Kanaal | Toon | Focus |
|--------|------|-------|
| Website | Professioneel-warm | Overtuigen, vertrouwen wekken |
| Documentatie | Helder, stap-voor-stap | Uitleggen, ondersteunen |
| AI Responses | Kort, behulpzaam | Informeren, assisteren |
| E-mail | Persoonlijk, scanbaar | Relatie onderhouden |
| Error messages | Geruststellend | Oplossing bieden |

## Checklist

Voordat je content publiceert:

- [ ] "Je/jij" consistent (nooit "u")
- [ ] Actieve zinnen
- [ ] Claims onderbouwd
- [ ] Geen onuitgelegd jargon
- [ ] Duidelijke CTA
- [ ] Spelling gecontroleerd

## Gebaseerd Op

Dit brandingpakket is ontwikkeld op basis van:
- Tone of Voice analyse van Exact Online (januari 2026)
- Site review van praatmetjeboekhouding.nl (januari 2026)
- Positionering als "natuurlijke uitbreiding" van Exact Online

De 80/20 regel: 80% Exact Online stijl voor herkenbaarheid, 20% eigen twist voor AI-interactie.
