Using the API documentation, you will be able to quickly integrate your application with Exact Online. You can use the following API: Exact Online REST API.

## Development Tips

### Explore Exact Online

- Use a demo administration of Exact Online for testing
- Sign up for a trial customer account
- Consult the Exact Online help documentation

### Use OAuth Registration

- Exact Online offers OAuth2.0
- See "Using OAuth 2.0 to access Exact Online API"

### Explore the API

- Use the free POSTMAN API client tool
- See the Reference documentation
