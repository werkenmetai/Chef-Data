## SEO Checklist per Template

Voor elk artikel, controleer:

### Titel
- [ ] Bevat primair keyword
- [ ] Max 60 karakters
- [ ] Wekt nieuwsgierigheid of belooft waarde

### Meta Description
- [ ] Bevat primair keyword
- [ ] Max 155 karakters
- [ ] Bevat CTA of belofte

### Content
- [ ] Keyword in eerste 100 woorden
- [ ] H2's bevatten secundaire keywords
- [ ] Minimaal 1000 woorden (tenzij nieuws)
- [ ] Afbeeldingen met alt-tekst
- [ ] Interne links (min. 3)
- [ ] Externe link naar autoritaire bron (1-2)
- [ ] FAQ sectie waar passend

### Technisch
- [ ] URL is kort en keyword-rijk
- [ ] Canonical tag aanwezig
- [ ] Open Graph tags
- [ ] Schema markup (Article/HowTo/FAQ)

---

## Tone of Voice Reminder

Bij elk artikel, check:

- ‚úÖ Spreek de lezer aan met "je"
- ‚úÖ Leg jargon uit of vermijd het
- ‚úÖ Begin met het probleem, niet de oplossing
- ‚úÖ Gebruik concrete voorbeelden
- ‚úÖ Eindig met een duidelijke volgende stap
- ‚ùå Geen lege superlatieven ("fantastisch", "revolutionair")
- ‚ùå Geen onnodige Engelse termen
- ‚ùå Niet te salesy in de body text
